# 电影感字幕生成器

一个基于Web的工具，允许用户上传图片并实时添加电影风格的字幕效果。

## 功能特性

- **实时预览** - 上传图片后可实时预览字幕效果，无需点击生成按钮
- **拖拽上传** - 支持拖拽或点击上传图片文件
- **字幕定制** - 可调整字幕区域高度、字体大小、粗细、样式、颜色等
- **多行字幕** - 支持多行字幕输入，每行字幕将生成一个新的堆叠层
- **即时下载** - 预览区域下方有独立的下载按钮，可直接下载生成的带字幕图片
- **响应式设计** - 适配不同屏幕尺寸

## 使用方法

1. 点击或拖拽图片到上传区域
2. 在字幕文本框中输入对话内容（每行将生成一个新的堆叠层）
3. 调整字幕设置参数（高度、字体、颜色等）
4. 预览区域将实时显示带字幕的图片效果
5. 点击"下载图片"按钮保存结果

## 安装和运行

1. 克隆或下载项目文件
2. 使用现代浏览器打开 `index.html` 文件
3. 无需额外安装或配置

或者使用本地服务器（如Python的http.server）来运行：

```bash
# Python 3
python -m http.server 8000

# 然后访问 http://localhost:8000
```

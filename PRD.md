# 电影感字幕生成器 - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目名称
电影感字幕生成器

### 1.2 项目描述
一个基于Web的工具，允许用户上传图片并添加电影风格的字幕效果。用户可以实时预览字幕效果，并下载生成的带字幕图片。

### 1.3 目标用户
- 内容创作者
- 社交媒体用户
- 电影爱好者
- 设计师

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 图片上传
- 支持拖拽上传图片
- 支持点击上传图片
- 支持常见图片格式（JPEG, PNG, WEBP等）

#### 2.1.2 字幕效果设置
- 字幕区域高度调整（10%-40%）
- 字体大小调整（2%-10%）
- 字体粗细选择（常规、加粗、特粗）
- 字体样式选择（多种中英文字体）
- 文字颜色选择
- 轮廓/阴影颜色选择

#### 2.1.3 实时预览
- 上传图片后立即显示
- 调整字幕参数时实时更新预览
- 输入字幕文本时实时更新预览

#### 2.1.4 字幕文本输入
- 支持多行字幕输入
- 每行字幕将生成一个新的堆叠层

#### 2.1.5 下载功能
- 生成带字幕的图片后可下载
- 保存为JPG格式

### 2.2 界面布局

#### 2.2.1 上传区域
- 拖拽上传区域
- 上传提示文字

#### 2.2.2 实时预览区域
- 显示处理后的图片
- 不再显示原图预览

#### 2.2.3 下载按钮
- 位于预览框下方
- 独立于上传区域，避免误操作

#### 2.2.4 字幕设置面板
- 各种字幕参数调节器
- 实时响应参数变化

#### 2.2.5 字幕文本输入区
- 多行文本输入框
- 支持多行字幕输入

## 3. 非功能性需求

### 3.1 性能需求
- 实时预览响应时间 < 500ms
- 图片上传后立即显示
- 参数调整实时更新

### 3.2 兼容性需求
- 现代浏览器支持（Chrome, Firefox, Safari, Edge）
- 响应式设计，支持不同屏幕尺寸

### 3.3 用户体验需求
- 界面简洁直观
- 操作流程清晰
- 提供即时反馈

## 4. 技术实现

### 4.1 技术栈
- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API（图片处理）

### 4.2 核心算法
- 图片分层处理算法
- 字幕区域计算
- 文字渲染效果

## 5. 用户流程

1. 用户访问网页
2. 拖拽或点击上传图片
3. 输入字幕文本
4. 调整字幕参数（实时预览更新）
5. 点击下载按钮保存结果

## 6. UI/UX设计要点

- 采用暗色主题，突出图片预览
- 玻璃态设计风格
- 响应式布局
- 清晰的视觉层次

## 7. 版本迭代

### 7.1 当前版本特性
- 实时预览功能
- 独立下载按钮（位于预览框外）
- 移除原图预览功能
- 优化用户体验
- 修复按钮遮挡问题

### 7.2 后续可能的迭代
- 批量处理功能
- 更多字幕样式选项
- 自定义模板功能
